How Do I Add Menu Items?

This guide is for restaurant staff and menu editors who use SafeEats to keep menu items and allergen tags up to date.

Quick overview

In SafeEats, you can add items one by one, or upload a menu and edit the generated items. For each item you’ll:

  • Fill in name, description, and price.
  • Optionally paste an ingredients list and use Parse with AI.
  • Confirm or adjust allergen and dietary checkboxes.
  • Save your changes so they appear on the restaurant dashboard.

Option 1 – Add items manually

  1. 1 From the dashboard, click Create New Items.
  2. 2 On the menu page, use the form labeled Menu Item #1:
    • Item Name*: short, clear name customers see.
    • Description*: a one‑to‑two line description of the dish.
    • Price*: type numbers only – it will be formatted as $12.99.
  3. 3 In the Ingredients box, you can paste the ingredient list from a label or recipe (optional).
  4. 4 Use the Dietary Options and Allergens checkboxes to mark the item correctly.
  5. 5 Click the appropriate save button (for your view) to send items to the backend.

You can add multiple items at once using multiple forms before saving them.

Option 2 – Upload a menu file

  1. 1 From the restaurant dashboard, click Create New Items.
  2. 2 Look for the upload control and choose a PNG, JPEG, or PDF of your menu.
  3. 3 Wait for “Extracting items…” to finish. SafeEats will create a form for each detected menu item.
  4. 4 Go through each generated item to fix names, descriptions, and prices where needed.

Always review items after an upload – the AI may miss special formatting or unusual menu layouts.

Using “Parse with AI” on ingredients

  1. 1 Paste the ingredient list into the Ingredients box (for example: milk, eggs, wheat flour, butter).
  2. 2 Click Parse with AI.
  3. 3 SafeEats will:
    • Highlight Parsed allergens just under the button.
    • Update the allergen and dietary checkboxes below the form.
  4. 4 Double‑check the suggestions. If anything looks wrong, simply toggle the checkboxes manually.
Tip: If parsing fails, you’ll see a message like “Could not parse ingredients. Please check the text and try again, or set allergens manually.” You can always fall back to manual checkboxes.

Saving and confirming items

  • When you’re done editing a row, use the provided Save/Update controls to commit that item.
  • For bulk imports, use the “Add All Items” action (if available) to send multiple items at once.
  • Return to the restaurant dashboard to confirm the items appear as expected.

Common issues

  • Price looks wrong: make sure you’re typing only digits (e.g., 1299 for $12.99); the field will format it automatically.
  • AI says “None detected” but you see allergens: toggle the correct allergen boxes manually (for example, turn on Milk, Eggs, Wheat).
  • Upload shows strange items: delete incorrect rows and add those dishes manually instead.
